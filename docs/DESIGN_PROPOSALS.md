# WidgetTodo Design Proposals

> 10種類のユニークなデザイン案 - どれを採用しますか？

---

## 1. 🌙 Midnight Aurora（ミッドナイトオーロラ）

**コンセプト**: 夜空に輝くオーロラをイメージした、ダークでミステリアスなデザイン

```
Primary:     #0D0D1A (深い夜空)
Secondary:   #1A1A2E (ダークネイビー)
Accent:      #00D9FF (シアンオーロラ)
Accent2:     #FF00E5 (マゼンタオーロラ)
Text:        #E0E0FF (淡いラベンダー)
Surface:     #16162A (カード背景)
```

**特徴**:
- グラデーション背景（オーロラ風のシアン→マゼンタ）
- ネオングロウエフェクト
- 角丸大きめ（16dp）
- フローティングカード with シャドウ
- アニメーション: タスク追加時に輝くエフェクト

**フォント**:
- 見出し: Orbitron（未来的）
- 本文: Noto Sans JP

**ウィジェット**: 半透明ダークガラス + オーロラボーダー

---

## 2. 🍃 Zen Garden（禅ガーデン）

**コンセプト**: 日本庭園の静けさと調和を表現した、落ち着いたミニマルデザイン

```
Primary:     #F5F2EB (和紙)
Secondary:   #E8E4DA (砂)
Accent:      #5C7A5C (苔)
Accent2:     #8B7355 (木)
Text:        #2C2C2C (墨)
Surface:     #FFFFFF (白)
```

**特徴**:
- 和紙テクスチャ背景
- 極限までシンプルなUI
- 余白を贅沢に使用
- 縦書きタイトルオプション
- 水墨画風のアイコン

**フォント**:
- 見出し: Zen Maru Gothic
- 本文: Noto Serif JP

**ウィジェット**: 和紙風背景 + 筆文字風チェックマーク

---

## 3. 🎮 Pixel Retro（ピクセルレトロ）

**コンセプト**: 80年代ゲーム機を彷彿とさせる、ノスタルジックなピクセルアート風

```
Primary:     #0F380F (ゲームボーイグリーン)
Secondary:   #306230 (ミディアムグリーン)
Accent:      #8BAC0F (ライトグリーン)
Accent2:     #9BBC0F (ハイライト)
Text:        #0F380F (ダークグリーン)
Surface:     #9BBC0F (スクリーン)
```

**特徴**:
- ピクセルフォント
- 8bit風アイコン
- CRTスキャンラインエフェクト
- チップチューン風の効果音
- ドット絵のチェックボックス

**フォント**:
- 全体: Press Start 2P / DotGothic16

**ウィジェット**: ゲームボーイスクリーン風 + ピクセルボーダー

---

## 4. 🌸 Sakura Bloom（桜ブルーム）

**コンセプト**: 春の桜をモチーフにした、柔らかくフェミニンなデザイン

```
Primary:     #FFF5F5 (桜霞)
Secondary:   #FFE4E9 (淡桜)
Accent:      #FF6B9D (濃桜)
Accent2:     #C44569 (紅)
Text:        #4A3F55 (藤紫)
Surface:     #FFFFFF (白)
```

**特徴**:
- 桜の花びらアニメーション（背景に舞う）
- ソフトシャドウ
- 丸みを帯びたUI要素
- グラデーションボタン
- 花びら型のFAB

**フォント**:
- 見出し: Kosugi Maru
- 本文: M PLUS Rounded 1c

**ウィジェット**: 淡いピンクグラデーション + 桜アイコン

## 5. 🏭 Industrial（インダストリアル）

**コンセプト**: 工業デザインにインスパイアされた、無骨でストイックなデザイン

```
Primary:     #1C1C1C (鉄)
Secondary:   #2D2D2D (鋼)
Accent:      #FF6B00 (警告オレンジ)
Accent2:     #FFB800 (イエロー)
Text:        #E0E0E0 (アルミ)
Surface:     #252525 (パネル)
```

**特徴**:
- シャープなエッジ（角丸なし）
- 警告テープ風のアクセント
- モノスペースフォント
- ボルト/リベット風の装飾

**フォント**:
- 見出し: Bebas Neue

**ウィジェット**: メタルテクスチャ + オレンジアクセント

```
Accent2:     #95E1D3 (ソーダ)

```

**特徴**:
- 多色グラデーション
- バウンスアニメーション
- ぷっくりした3D風ボタン

**フォント**:

## 7. 📰 Editorial（エディトリアル）

**コンセプト**: 高級雑誌のようなエレガントで洗練されたタイポグラフィ重視のデザイン

```
Primary:     #FAFAFA (オフホワイト)
Secondary:   #F0F0F0 (ライトグレー)
Accent:      #000000 (ブラック)
Accent2:     #C9A962 (ゴールド)
Text:        #1A1A1A (ディープブラック)
Surface:     #FFFFFF (ピュアホワイト)
```

**特徴**:
- 大きなセリフフォント
- 黄金比レイアウト
- 細いボーダーライン
- ドロップキャップ
- モノクロ + ゴールドアクセント

**フォント**:
- 見出し: Playfair Display
- 本文: Source Serif Pro / Noto Serif JP

**ウィジェット**: クリーンホワイト + エレガントタイポグラフィ

## 8. 🌊 Ocean Depth（オーシャンデプス）

**コンセプト**: 深海の神秘と静けさを表現した、没入感のあるデザイン

```
Primary:     #0A1628 (深海)
Secondary:   #1A2F4E (ミッドナイトブルー)
Accent:      #00F5D4 (バイオルミネセンス)
Accent2:     #00BBF9 (浅瀬)
Text:        #E0F7FF (泡)
Surface:     #132238 (海底)
```

**特徴**:
- 波紋アニメーション（タップ時）
- 泡のパーティクル
- 深度による色の変化
- 流れるようなトランジション
- 発光するアクセント

**フォント**:
- 見出し: Comfortaa
- 本文: Quicksand

**ウィジェット**: 深海ブルーグラデーション + 発光エフェクト

---

## 9. 🎨 Neo Brutalism（ネオブルータリズム）

**コンセプト**: 大胆な色使いと強いコントラスト、モダンブルータリストスタイル

```
Primary:     #FFFF00 (ビビッドイエロー)
Secondary:   #FF90E8 (ホットピンク)
Accent:      #000000 (ブラック)
Accent2:     #90F4FF (シアン)
Text:        #000000 (ブラック)
Surface:     #FFFFFF (ホワイト)
```

**特徴**:
- 太い黒ボーダー（4dp）
- ハードシャドウ（オフセット）
- 原色使用
- 意図的な「粗さ」
- ホバーでズレるエフェクト

**フォント**:
- 見出し: Space Mono
- 本文: Work Sans

**ウィジェット**: イエロー背景 + 黒太枠 + オフセットシャドウ

---

## 10. 🌿 Forest Calm（フォレストカーム）

**コンセプト**: 森林浴のような癒しと落ち着きを与える、ナチュラルなデザイン

```
Primary:     #F4F1DE (クリーム)
Secondary:   #E9E5D6 (サンドベージュ)
Accent:      #3D5A3D (フォレストグリーン)
Accent2:     #81B29A (セージ)
Text:        #3D405B (チャコール)
Surface:     #FDFCF7 (アイボリー)
```

**特徴**:
- オーガニックな形状（blob形状）
- 葉っぱのアイコン
- ナチュラルなテクスチャ
- スローなイージングアニメーション
- 木漏れ日エフェクト

**フォント**:
- 見出し: Lora
- 本文: Open Sans / Noto Sans JP

**ウィジェット**: クリーム背景 + グリーンアクセント + 葉アイコン

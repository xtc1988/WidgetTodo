# デザインスタイルパック一覧

1. **Nord Mist**
   - カラーパレット: ディープブルー、ミストグレー、スノーホワイト、アクセントにティール。
   - 特徴: 丸みのあるカード、柔らかな影、落ち着いた北欧テイスト。
   - ウィジェット背景: グラデーション(ブルー→グレー)にソフトなガラス感。

2. **Neon Minimal**
   - カラーパレット: ピュアブラックベース、シアン/マゼンタ/ライムのネオンアクセント。
   - 特徴: 細身フォント、ピクセルパーフェクトなライン、超高コントラスト。
   - ウィジェット背景: 黒ベース＋ネオンラインのボーダー。

3. **Paper Craft**
   - カラーパレット: オフホワイト、ベージュ、サンド、ネイビーブラック。
   - 特徴: 紙の質感やスキューモーフ風の浮き、セリフ体見出し。
   - ウィジェット背景: シボ感テクスチャ入りの淡色カード。

4. **Solar Dawn**
   - カラーパレット: アンバー、ピーチ、サーモン、ディープパープル。
   - 特徴: 朝焼けグラデのピル形状、柔らかアニメーション。
   - ウィジェット背景: 斜めに走るアンバー→ピーチグラデ。

5. **Tech Line**
   - カラーパレット: モノクロ＋エレクトリックライム。
   - 特徴: 幾何学ライン、凝縮フォント、フラットでシャープ。
   - ウィジェット背景: グリッドパターンにラインアクセント。

6. **Pastel Blocks**
   - カラーパレット: ミント、ラベンダー、コーラル、アイスブルー。
   - 特徴: ブロック状のセクション、丸み、ポップな印象。
   - ウィジェット背景: 色分けカードを並べたパッチワーク風。

7. **Brutalist Grid**
   - カラーパレット: ピュアホワイト、ジェットブラック、赤の差し色。
   - 特徴: 太いグリッド、ゴシック大文字、意図的に無骨。
   - ウィジェット背景: 格子状の線と大きな数字で日付表示。

8. **Glass Morph**
   - カラーパレット: 透過ホワイト、ライトシアン、チタングレー。
   - 特徴: フロストカード、ニューモーフ気味のボタン、柔らか影。
   - ウィジェット背景: ブラーをかけた壁紙上に浮かぶカード。

9. **Monoline Outline**
   - カラーパレット: ソフトホワイト、チャコール、オレンジアクセント。
   - 特徴: アウトラインカード、線アイコン、余白を大きくとる。
   - ウィジェット背景: 細い枠線でセクションを分けるシンプル構成。

10. **Dynamic Material**
    - カラーパレット: Material Youの動的カラーをフル活用。大きなタイポとアダプティブシェイプ。
    - 特徴: Androidシステム設定と連動し、季節感や壁紙色で変化。
    - ウィジェット背景: Dynamic Colorベースのシェイプ重ね。

各スタイルは`StylePack`としてカラートークン、タイポグラフィ、カード形状、チェックボックスデザインを保持し、アプリ本体とウィジェットに同じテーマを適用する。

## 📄 詳細な説明（4000文字以内）

```
WidgetTodoは、シンプルさと使いやすさを追求したタスク管理アプリです。
余計な機能を排除し、本当に必要なものだけを厳選しました。

【特徴】

✓ ミニマルなデザイン
禅的な落ち着いた配色とシンプルなUIで、タスク管理に集中できます。
余計な装飾は一切ありません。

✓ 高速起動
アプリを開けばすぐにタスクが確認できます。
無駄な読み込み時間はありません。

✓ ウィジェット対応
ホーム画面にウィジェットを配置すれば、アプリを開かなくても
やることが一目で確認できます。

✓ 直感的な操作
・タスクの追加：画面右下の「+」ボタンをタップ
・タスクの完了：チェックマークをタップ
・元に戻す：完了直後に表示される「元に戻す」をタップ

【こんな人におすすめ】

・シンプルなタスク管理アプリを探している
・複雑な機能は不要で、基本的なTODOリストがあれば十分
・ウィジェットでホーム画面からタスクを確認したい
・広告やサブスクリプションのないアプリが欲しい
・ミニマルなデザインが好き

【主な機能】

📌 タスクの追加・管理
新しいタスクを簡単に追加し、完了したらチェックマークで完了できます。

📌 Undoサポート
間違って完了してしまったタスクも、すぐに元に戻せます。

📌 ホーム画面ウィジェット
アプリを開かなくても、ホーム画面でタスクを確認できます。

📌 オフライン動作
インターネット接続不要。すべてのデータは端末内に保存されます。

📌 プライバシー重視
個人情報の収集は一切行いません。データは端末内のみに保存されます。

【デザイン理念】

「静寂」- タスクがないときの心の平穏
「シンプル」- 本当に必要な機能だけ
「直感的」- 説明なしで使える操作性

【開発者より】

WidgetTodoは、複雑化した現代のタスク管理アプリへのアンチテーゼとして開発されました。
プロジェクト管理、タグ付け、優先順位、リマインダー、コラボレーション...
これらの機能は素晴らしいものですが、時として「やることを書き留めておく」という
本来の目的を見失わせてしまいます。

WidgetTodoは、その原点に立ち返ります。
シンプルに、美しく、速く。
それがこのアプリの目指すものです。

【サポート】

このアプリは個人情報を収集しません。

詳細はプライバシーポリシーをご確認ください。
```
